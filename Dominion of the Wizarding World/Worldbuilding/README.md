# Worldbuilding Reference Files

This folder contains all the foundational reference materials for the Dominion of the Wizarding World.

## Core World Systems
- **Power_System.txt** - Complete progression system (Knight ranks, Wizard ranks, Laws, Domains)
- **Schools_of_Knowledge.txt** - Wizard education system and specialization paths
- **World_Politics.txt** - Council structure, void campaigns, wizard society hierarchy

## Character References
- **Knight_Names.txt** - Noble houses, common knights, MC's family (House Malaroth)
- **Wizard_Names.txt** - Wizard naming conventions, school names, cosmic titles

## Magic Systems
- **Warlock_System.txt** - Creature bonding mechanics, Chimera fusion (MC's unique path)
- **Insect_Tamer_System.txt** - Hive mode control, swarm mechanics

## Story Planning
- **Story_Arcs.txt** - Complete saga outline from transmigration to transcendence

## Usage Notes
- These files are CANONICAL - treat as absolute truth for the world
- When writing chapters, cross-reference these files for consistency
- Character names and power levels are established here
- Magic system mechanics are fully defined and balanced

## Key Concepts to Remember
1. Knights cannot sense mana - body cultivation only
2. Wizards radiate deadly energy - cannot live with mortals
3. MC has AI chip fused with soul - unique advantage
4. World follows "survival of the fittest" philosophy
5. Rank progression is extremely difficult and dangerous
